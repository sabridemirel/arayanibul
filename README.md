# Mobil Uygulama Projesi

## Proje Yapısı
- `mobile/` - React Native uygulaması
- `backend/` - .NET Core Web API

## Özellikler
- Email/şifre ile kayıt ve giriş
- Google ve Meta (Facebook) ile sosyal giriş
- Misafir kullanıcı girişi
- JWT tabanlı kimlik doğrulama
